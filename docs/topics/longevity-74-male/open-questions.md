# Open Questions

Questions this repository should keep revisiting as new evidence appears:

- Which interventions improve function and resilience specifically in adults over 70?
- How should older adults balance muscle preservation with weight loss or metabolic therapies?
- Do rapalogs improve clinically meaningful aging outcomes in humans?
- Are any senolytic strategies ready outside clinical trials?
- Which biomarkers are actually actionable rather than merely interesting?
- What screening intensity maximizes benefit and minimizes overdiagnosis in healthy older men?
- Which combinations of exercise, nutrition, and medications produce additive benefit versus unnecessary burden?
