# Biomarkers & Measurement

!!! warning
    Biomarkers should be interpreted by qualified clinicians in clinical context.

## Practical clinical markers to track

### Cardiometabolic

- Blood pressure, including orthostatic symptoms if relevant
- Lipid panel, preferably including ApoB and Lp(a) at least once
- HbA1c, fasting glucose, fasting insulin where appropriate
- Kidney and liver function
- hs-CRP when clinically useful

### Function and body composition

- Grip strength
- Gait speed
- Chair-stand performance
- Balance tests
- DEXA for bone density, and body composition if useful
- Waist circumference

### Nutrition and deficiency risk

- Vitamin B12, especially if using metformin or with neuropathy/cognitive symptoms
- Vitamin D when clinically indicated
- Iron studies if anemia/fatigue is present

### Cognitive/sleep

- Sleep apnea screening when indicated
- Cognitive baseline if concerns arise

## Experimental / cautionary markers

- Epigenetic clocks
- Biological age panels
- Telomere length
- Broad inflammatory cytokine panels
- Consumer microbiome tests

These may be interesting but should not drive major interventions without stronger evidence.
