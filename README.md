# Longevity Research Library

A curated, continuously updated research repository focused on longevity enhancement for a 74-year-old male.

This repository is designed for evidence review and discussion with qualified clinicians. It is not medical advice.

## Web interface

This repo is structured for MkDocs Material. To preview locally after installing dependencies:

```bash
python3 -m pip install -r requirements.txt
mkdocs serve
```

Then open http://127.0.0.1:8000

## Main sections

- `docs/topics/longevity-74-male/` — core research dossier
- `docs/updates/` — dated research updates
- `policies/selection-policy.md` — what counts as high-quality evidence
- `sources/sources.yaml` — trusted query/feed configuration
- `prompts/` — Hermes prompts for scheduled updates and synthesis

## Safety stance

The repository prioritizes interventions with human evidence, plausible benefit, tolerable risk, and relevance to older men. It flags, rather than promotes, speculative or high-risk interventions.
